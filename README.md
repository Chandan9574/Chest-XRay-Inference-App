# Chest X-Ray Inference App

A TensorFlow Lite based chest X-ray classification application built using Streamlit.

Features:
- COVID-19 detection
- Viral Pneumonia detection
- Normal chest X-ray detection
- Confidence score display

Tech Stack:
- Python
- TensorFlow Lite
- Streamlit
- NumPy

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py
